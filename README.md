# BAPATA - Simple E-Commerce Backend (C++)

A console-based e-commerce backend application built with C++ as a college group project.

## Features
- User authentication (Register, Login, Edit Profile)
- Product catalog management
- Shopping cart (Add, Update, Remove items)
- Checkout & transaction history
- Admin panel for order status management

## Tech Stack
- Language: C++
- Storage: File-based (.txt)
- Architecture: OOP (Object-Oriented Programming)

## How to Run
1. Compile all `.cpp` files using g++: g++ MainApp.cpp user.cpp -o bapata
2. Run the executable: ./bapata.exe
